Project Purpose
This application is built as part of the Lumos technical evaluation task. It demonstrates a complete working flow including authentication, file upload, server-side storage, and generation of a shareable access link.
How to use this application
  1. Select a file using the upload button below.
  2. Click Upload to store the file on the server.
  3. A shareable link will be generated after successful upload.
  4. Anyone with the link can access the file directly.
